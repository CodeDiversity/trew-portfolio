Projects

Rate My Diy

Rate My DIY is a project that I worked on with a team of other student developers during Lambda School with. We used React JS and Apollo Client for the front-end, and for the back-end we used GraphQL, and Apollo Server and Prisma for our backend. For styling the app, we used a couple libraries for animations, but most of the CSS was done using pure node-sass/css. The premise of this app is to be a website where users can create an account, and share their favorite DIY home projects, additionally, other users can either up vote, or down vote the project and leave reviews about the projects.
